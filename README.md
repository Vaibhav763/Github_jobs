# Recruit_Plus
Designed Web Application for finding the Jobs Opening as per our
requirement by fetching latest jobs from Muse API.
<h1>Features </h1>
<h4>• Dynamic Search Bar</h4>
<h4>• Pagination</h4>
<h4>• Dark Light Mode Switching</h4>
<h4>• Used Lottie Animation</h4>
<h4>• MUSE API </h4>
